# helloworld-python
Hello, World! in Python

## Setup

```bash
pipenv shell
```

```bash
pipenv sync
```

## Build

```bash
make build
```

## Run

```bash
./dist/helloworld
```

### Download from GitHub releases and run it

```bash
curl -L -o helloworld https://github.com/wasabina67/helloworld-python/releases/download/v1/helloworld
```

```bash
chmod +x helloworld
```

```bash
./helloworld
```
